# UI

Nodes for controlling user interface elements in Range Game Engine: text objects, progress bars, object colors, and overlay scenes.

## Nodes

| Node | Type | Description |
|------|------|-------------|
| [Set Text](set-text.md) | Action | Set the text of a font object (KX_FontObject) |
| [Set Bar](set-bar.md) | Action | Scale an object on X to represent a fill percentage (health bar, etc.) |
| [Set Color](set-color.md) | Action | Change the RGBA color of an object |
| [Overlay Scene](overlay-scene.md) | Action | Add, remove, or toggle an overlay scene (HUD, menu, etc.) |
