# MATH Category

Nodes for mathematical, vector, random, and object distance/direction operations.

## Available Nodes

| Node | Type | Description |
|------|------|-------------|
| [Math](math.md) | Data | Adds, subtracts, multiplies, or divides two values |
| [Get Distance](get-distance.md) | Data | Distance between two objects (XY or XYZ) |
| [Get Direction](get-direction.md) | Data | Component of the normalized direction between two objects |
| [Vector Math](vector-math.md) | Data | 10 vector operations on 3D vectors |
| [Object Position](object-position.md) | Data | World position component of an object |
| [Random Float](random-float.md) | Data | Random float in range [min, max] |
| [Random Int](random-int.md) | Data | Random integer in range [min, max] |
| [Random Bool](random-bool.md) | Data | Random boolean with configurable probability |
| [Random Pick](random-pick.md) | Data | Selects a random element from a list |
| [Random Chance](random-chance.md) | Branch | Branch by probability |
| [Set Random Seed](set-random-seed.md) | Action | Sets the random number generator seed |
