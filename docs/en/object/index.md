# OBJECT Category

Nodes for manipulating scene objects: visibility, transforms, parent hierarchy, adding/removing objects, and scene control.

## Available Nodes

| Node | Type | Description |
|------|------|-------------|
| [Set Visibility](set-visibility.md) | Action | Shows or hides the current object |
| [Look At](look-at.md) | Action | Rotates the object to face a target (slerp) |
| [Transform](transform.md) | Action | Moves, rotates, or scales the object |
| [Add Object](add-object.md) | Action | Instances an object into the scene |
| [End Object](end-object.md) | Action | Destroys the current object |
| [Replace Mesh](replace-mesh.md) | Action | Replaces the object's mesh |
| [Set Parent](set-parent.md) | Action | Sets the object's parent |
| [Remove Parent](remove-parent.md) | Action | Removes the object's parent |
| [Send Message](send-message.md) | Action | Sends a BGE message (native system) |
| [Restart Game](restart-game.md) | Action | Restarts the entire game |
| [Scene Switch](scene-switch.md) | Action | Switches to another scene |
