# LOGIC Category

Nodes for conditional flow control, comparison, and BGE property manipulation.

## Available Nodes

| Node | Type | Description |
|------|------|-------------|
| [Branch](branch.md) | Branch | Generic if/else branch based on a condition |
| [Add To Property](add-to-property.md) | Action | Adds a value to a BGE property |
| [Toggle](toggle.md) | Action | Toggles a boolean BGE property |
| [If Property](if-property.md) | Branch | Compares a BGE property to a value using 6 operators |
| [Compare](compare.md) | Data | Returns a boolean by comparing two values |
| [Logic Gate](logic-gate.md) | Data | Logical operation between two booleans (AND, OR, NOT, etc.) |
| [Random](random.md) | Data | Random float [0.0, 1.0) — legacy node |
