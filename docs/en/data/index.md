# DATA Category

Nodes for reading, writing, and manipulating data: BGE properties, strings, and system time.

## Available Nodes

| Node | Type | Description |
|------|------|-------------|
| [Get Property](get-property.md) | Data | Reads a BGE property from the object |
| [Set Property](set-property.md) | Action | Writes a value to a BGE property |
| [Timer](timer.md) | Data | Returns the absolute system time |
| [Concat String](concat-string.md) | Data | Concatenates two strings with a separator |
| [Format String](format-string.md) | Data | Formats a template string with up to 4 values |
| [Number To String](number-to-string.md) | Data | Converts a number to a string with prefix/suffix |
| [String Length](string-length.md) | Data | Returns the length of a string |
| [String Contains](string-contains.md) | Branch | Detects whether a string contains another |
| [String Equal](string-equal.md) | Branch | Compares two strings with case-sensitive option |
