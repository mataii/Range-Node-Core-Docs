# Plantillas de inicio

Configuraciones de proyecto listas para usar. Cada plantilla te da la configuración exacta de escena y el NodeTree para tener un prototipo funcionando en minutos — sin explicaciones.

Para entender *por qué* funciona cada paso, sigue el tutorial correspondiente en [Sistemas de juego](../guias/sistemas/index.md).

| Plantilla | Nodo principal | Enfoques | Tiempo de configuración |
|-----------|---------------|----------|------------------------|
| [Prototipo FPS](prototipo-fps.md) | FPS Tools | Primera persona, Tercera persona | ~5 min |

Más plantillas (Vehicle, Drone, Personaje RPG) están planificadas para versiones futuras.
