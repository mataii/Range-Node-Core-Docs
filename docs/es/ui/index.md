# UI

Nodos para controlar elementos de interfaz de usuario en Range Game Engine: textos, barras de progreso, colores de objetos y escenas de overlay.

## Nodos

| Nodo | Tipo | Descripción |
|------|------|-------------|
| [Set Text](set-text.md) | Acción | Establece el texto de un objeto de fuente (KX_FontObject) |
| [Set Bar](set-bar.md) | Acción | Escala un objeto en X para representar un porcentaje (barra de vida, etc.) |
| [Set Color](set-color.md) | Acción | Cambia el color RGBA de un objeto |
| [Overlay Scene](overlay-scene.md) | Acción | Añade, elimina o alterna una escena superpuesta (HUD, menú, etc.) |
