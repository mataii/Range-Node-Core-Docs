# Categoría OBJECT

Nodos para manipular objetos de la escena: visibilidad, transformaciones, jerarquía de padres, añadir/eliminar objetos y control de escenas.

## Nodos disponibles

| Nodo | Tipo | Descripción |
|------|------|-------------|
| [Set Visibility](set-visibility.md) | Acción | Muestra u oculta el objeto actual |
| [Look At](look-at.md) | Acción | Rota el objeto para mirar hacia un objetivo (slerp) |
| [Transform](transform.md) | Acción | Mueve, rota o escala el objeto |
| [Add Object](add-object.md) | Acción | Instancia un objeto en la escena |
| [End Object](end-object.md) | Acción | Destruye el objeto actual |
| [Replace Mesh](replace-mesh.md) | Acción | Reemplaza la malla del objeto |
| [Set Parent](set-parent.md) | Acción | Establece el padre del objeto |
| [Remove Parent](remove-parent.md) | Acción | Elimina el padre del objeto |
| [Send Message](send-message.md) | Acción | Envía un mensaje BGE (sistema nativo) |
| [Restart Game](restart-game.md) | Acción | Reinicia el juego completo |
| [Scene Switch](scene-switch.md) | Acción | Cambia a otra escena |
