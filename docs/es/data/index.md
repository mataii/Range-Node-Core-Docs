# Categoría DATA

Nodos para leer, escribir y manipular datos: propiedades BGE, cadenas de texto y tiempo de sistema.

## Nodos disponibles

| Nodo | Tipo | Descripción |
|------|------|-------------|
| [Get Property](get-property.md) | Dato | Lee una propiedad BGE del objeto |
| [Set Property](set-property.md) | Acción | Escribe un valor en una propiedad BGE |
| [Timer](timer.md) | Dato | Devuelve el tiempo absoluto del sistema |
| [Concat String](concat-string.md) | Dato | Concatena dos cadenas con separador |
| [Format String](format-string.md) | Dato | Formatea una plantilla con hasta 4 valores |
| [Number To String](number-to-string.md) | Dato | Convierte un número a cadena con prefijo/sufijo |
| [String Length](string-length.md) | Dato | Devuelve la longitud de una cadena |
| [String Contains](string-contains.md) | Rama | Detecta si una cadena contiene otra |
| [String Equal](string-equal.md) | Rama | Compara dos cadenas con opción case-sensitive |
