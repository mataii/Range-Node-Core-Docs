# Categoría LOGIC

Nodos para control de flujo condicional, comparación y manipulación de propiedades BGE.

## Nodos disponibles

| Nodo | Tipo | Descripción |
|------|------|-------------|
| [Branch](branch.md) | Rama | Bifurcación genérica si/sino basada en una condición |
| [Add To Property](add-to-property.md) | Acción | Suma un valor a una propiedad BGE |
| [Toggle](toggle.md) | Acción | Alterna un booleano en una propiedad BGE |
| [If Property](if-property.md) | Rama | Compara una propiedad BGE con un valor usando 6 operadores |
| [Compare](compare.md) | Dato | Devuelve un booleano comparando dos valores |
| [Logic Gate](logic-gate.md) | Dato | Operación lógica entre dos booleanos (AND, OR, NOT, etc.) |
| [Random](random.md) | Dato | Float aleatorio [0.0, 1.0) — nodo legacy |
