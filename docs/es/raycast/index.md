# RAYCAST

Nodos para lanzar rayos en la escena de Range Game Engine y detectar colisiones, objetos, terreno y portales.

## Nodos

| Nodo | Tipo | Descripción |
|------|------|-------------|
| [Raycast](raycast.md) | Acción (rama) | Lanza un rayo en una dirección local o hacia un objeto |
| [General Raycast](general-raycast.md) | Acción (rama) | Rayo entre dos puntos arbitrarios en espacio mundo |
| [Raycast Result](raycast-result.md) | Dato | Lee el resultado del último raycast por ID |
| [Ground Check](ground-check.md) | Acción (rama) | Comprueba si el objeto está en el suelo (rayo descendente) |
| [Mouse Pick](mouse-pick.md) | Acción (rama) | Rayo desde la cámara activa a través del cursor del ratón |
| [Portal Teleport](portal-teleport.md) | Acción | Teletransporta un objeto entre dos portales al estilo Portal |
